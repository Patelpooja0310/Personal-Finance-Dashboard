# 💰 Personal Finance Dashboard

## 📌 Overview
A Streamlit-based Personal Finance Dashboard with MySQL, Machine Learning, and Data Visualization.

## ✨ Features
- User Authentication
- Income & Expense Tracking
- Interactive Dashboard
- Financial Reports
- Data Visualization
- Machine Learning Insights

## 🛠️ Tech Stack
- Python
- Streamlit
- MySQL
- Plotly
- Scikit-learn
- bcrypt

## 🚀 How to Run
1. Clone the repository
2. Install the required packages:
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   streamlit run app.py
   ```
